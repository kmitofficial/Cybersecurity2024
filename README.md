# Instructions
## :syringe: SQLi attack + prevention
### Steps to run server
1. Change to **server** directory

    `cd SQLi/server`

2. Before first run, install all node modules (needs to be run only once)

    `npm install`

3. Run server

    `npm start`


### Steps to run client
1. Change to **client** directory

    `cd SQLi/client`

2. Before first run, install all node modules (needs to be run only once)

    `npm install`

3. Run client

    `npm start`
